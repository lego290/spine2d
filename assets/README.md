# assets/

This directory is intended for **local development assets** (not committed by default).

If you want to run upstream Spine example smoke tests or render real example skeletons, fetch the upstream data into:

- `assets/spine-runtimes/`

See:

- `scripts/fetch_spine_runtimes_examples.py`
- `scripts/import_spine_runtimes_examples.zsh`

Upstream examples known to include Spine 4.3 physics constraints:

- `cloud-pot/export/cloud-pot.json`
- `sack/export/sack-pro.json`
- `celestial-circus/export/celestial-circus-pro.json`
- `snowglobe/export/snowglobe-pro.json`
